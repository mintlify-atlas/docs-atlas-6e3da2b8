---
title: "Keyboard shortcuts"
description: "View and customize every keyboard shortcut in Chatterino7 across four configurable categories."
---

Chatterino7 has a fully configurable hotkey system. Every default keybinding can be changed, and you can add additional bindings for any action. Hotkeys are organized into four categories based on where they apply.

## Configuring hotkeys

<Steps>
  <Step title="Open Settings">
    Press `Ctrl+P` to open Settings.
  </Step>
  <Step title="Navigate to Keyboard shortcuts">
    Click **Keyboard shortcuts** in the left sidebar.
  </Step>
  <Step title="Edit a hotkey">
    Double-click any row to edit its key sequence. Enter the new key combination and click **Save**.
  </Step>
  <Step title="Add a custom binding">
    Click **Add** to create a new hotkey entry. Select the category, action, and key sequence.
  </Step>
  <Step title="Reset to defaults">
    Click **Reset to defaults** to restore all hotkeys to their built-in values.
  </Step>
</Steps>

<Warning>
Renaming or deleting a hotkey that shares a name with a default can cause the default to be re-added on next launch. If you want to disable a default keybinding, clear its key sequence rather than deleting the entry.
</Warning>

## Running commands with a hotkey

The **Run a command** action in the Split category lets you bind a key sequence to any custom command you have defined in Settings → Commands.

<Steps>
  <Step title="Create a command">
    Define a custom command in **Settings → Commands**. For example, `/gg` expands to `gg everyone!`.
  </Step>
  <Step title="Add a hotkey">
    In **Settings → Keyboard shortcuts**, add a new hotkey in the **Split** category with the action **Run a command**.
  </Step>
  <Step title="Set the command argument">
    In the argument field, enter the command trigger exactly as you named it (for example `/gg`).
  </Step>
</Steps>
